Configurar tu perfil

Cómo añadir una foto de perfil

Para añadir o cambiar una foto de perfil de TikTok, haz lo siguiente:

  1. En la aplicación TikTok, pulsa Perfil en la parte inferior.
  2. Pulsa Editar perfil.
  3. Pulsa Cambiar foto.
  4. Selecciona Hacer una foto o Cargar foto para usar una del dispositivo.
  5. Recorta la foto o el video que hayas seleccionado y pulsa Guardar.
Nota: Las fotos de perfil dejarán de estar visibles después de configurar un video de perfil. Las fotos deben ser de al menos 20 × 20 píxeles para poder cargarlas.

Cambio del apodo

En TikTok, el apodo es el nombre que aparece en tu cuenta, además del nombre de usuario. Tu apodo se muestra en varias partes de TikTok, incluido el feed «Para ti», la pestaña «Amigos», los comentarios y los mensajes directos. Todas las cuentas de TikTok deben tener un apodo.

Para cambiar tu apodo, haz lo siguiente:

  1. En la aplicación TikTok, pulsa Perfil en la parte inferior.
  2. Pulsa Editar perfil.
  3. Pulsa Nombre.
  4. Escribe el apodo deseado. Puedes escribir un máximo de 30 caracteres. Ten en cuenta que los apodos deben cumplir nuestras Normas de la comunidad.
  5. Pulsa Guardar.
Nota: El apodo solo se puede cambiar una vez cada 7 días.

Cambiar el nombre de usuario

El nombre de usuario en TikTok es el nombre que se muestra en tu perfil y en el enlace de tu perfil. También puede usarse para buscar o etiquetar una cuenta. Cada cuenta debe tener un nombre de usuario.

Para cambiar tu nombre de usuario en TikTok, haz lo siguiente:

  1. En la aplicación TikTok, pulsa Perfil en la parte inferior.
  2. Pulsa Editar perfil.
  3. Escribe el nombre de usuario deseado.
  4. Pulsa Guardar.

Ten en cuenta lo siguiente:

  • El nombre de usuario solo se puede cambiar una vez cada 30 días.
  • Si cambias tu nombre de usuario, no podrás añadir ese nombre de usuario en una cuenta durante un breve período de tiempo.
  • Cambiar el nombre de usuario también cambiará el enlace del perfil.
  • El nombre de usuario solo puede contener letras, números, guiones bajos y puntos. No obstante, no puedes añadir un punto al final del nombre de usuario.
Nota: Puedes contactarnos a través de la función Informar de un problema si quieres cambiar el nombre de usuario de tu cuenta verificada.

Cómo añadir un enlace a tu perfil de TikTok

Puedes añadir a tu perfil un enlace a cualquiera de tus otras cuentas de redes sociales. Así podrás compartir contenido y conectar con más gente fuera de TikTok.

Para añadir una cuenta de redes sociales a tu perfil de TikTok:

  1. En la aplicación TikTok, pulsa Perfil en la parte inferior.
  2. Pulsa Editar perfil.
  3. Junto a los enlaces, pulsa Añadir.
  4. Elige una cuenta de redes sociales que añadir (Lemon8, Instagram o YouTube) y sigue los pasos de inicio de sesión para vincularla.
Nota: Esta función no está disponible actualmente en todo el mundo.

Puedes añadir a tu perfil de TikTok un enlace a tu sitio web si tienes al menos 1000 seguidores o una cuenta de empresa registrada. Encontrarás más información sobre las cuentas de empresas registradas en el Centro de ayuda de TikTok para empresas.

Para añadir un sitio web a tu perfil de TikTok:

  1. En la aplicación TikTok, pulsa Perfil en la parte inferior.
  2. Pulsa Editar perfil.
  3. Junto a los enlaces, pulsa Añadir.
  4. Sigue los pasos para vincular tu sitio web.
Cambiar los ajustes de idioma

Puedes cambiar el idioma en el que se muestra la aplicación TikTok en cualquier momento.

Para cambiar los ajustes de idioma, haz lo siguiente:

  1. En la aplicación TikTok, pulsa Perfil en la parte inferior.
  2. Pulsa el botón Menú ☰ en la parte superior y selecciona Ajustes y privacidad.
  3. Pulsa Idioma.
  4. Pulsa Idioma de la aplicación y, después, selecciona un idioma de la lista.
  5. Pulsa Listo.
